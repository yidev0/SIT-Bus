# SIT Bus
SITのバス時刻表アプリ<br>

## 機能・ロードマップ
#### 時刻表表示
 - [x] スクールバス（大宮）
 - [x] スクールバス（岩槻）
 - [x] シャトルバス
#### iOSの機能
 - [x] ダークモード（アプリ・アイコン）
 - [x] ウィジェット（ホーム画面）
 - [ ] ウィジェット（ロック画面）
 - [ ] 通知
 - [ ] ライブアクティビティ
 - [x] ショートカット（次のバスを取得）
 - [ ] ショートカット（その他）
#### 言語
 - [x] 日本語
 - [x] 英語
 - [ ] 中国語（簡体）
 - [ ] フランス語
#### デバイス
 - [x] iPhone
 - [ ] Apple Watch
 - [ ] Android（未定・skip.toolsを利用予定）
